# Mario Sánchez Portfolio 🚀

Professional Portfolio of Mario Sánchez — Front-end Developer specialized in React, TypeScript, and Tailwind CSS.

## 🔗 Live Demo
**[View Live Portfolio](https://mjsanchezr.github.io/Portafolio/)**

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **React 18** + **TypeScript** | Core application logic |
| **Tailwind CSS 3** | Modern styling and design |
| **Framer Motion** | Smooth animations and scroll effects |
| **Recharts** | Data visualization for project metrics |
| **React Hook Form** | Contact form management and validation |
| **Vite** | Build tool and optimized dev server |

## 📁 Project Structure

```
Portafolio/
├── public/              # Static assets (Favicon, Profile Image)
├── src/
│   ├── components/      # UI Sections (Hero, About, Projects, etc.)
│   ├── hooks/           # Business logic (useProjects, useScrollSpy)
│   ├── data/            # Dynamic content (portfolioData.json)
│   ├── types/           # TypeScript Interfaces
│   ├── utils/           # Helper functions
│   ├── App.tsx          # Root layout
│   └── index.css        # Global styles & Tailwind
└── (config files)       # Vite, TS, Tailwind, PostCSS
```

## 🏗️ Technical Highlights

- **Container/Presentation Pattern**: Logic is decoupled from UI using custom hooks.
- **Data-Driven UI**: Entire content is manageable via a single JSON file.
- **Modern Aesthetics**: Implements glassmorphism, animated blobs, and a premium dark theme.
- **Accessibility & SEO**: Semantic HTML, meta tags, and responsive design for all devices.

## 🚀 Installation & Setup

1. **Clone and Install**
```bash
git clone https://github.com/mjsanchezr/Portafolio.git
cd Portafolio
npm install
```

2. **Development**
```bash
npm run dev
# Open http://localhost:5173
```

3. **Build & Deploy**
```bash
npm run build    # Generates /dist
npm run deploy   # Deploys to GitHub Pages
```
